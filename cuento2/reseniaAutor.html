Este es un sistema de venta de gasolina, el acceso esta restringido mediante contraseña y solo pueden entrar los empleados registados

Una vez dentro se elige la fecha de un formulario, el tipo de gasolina el cual ya esta guardado en la base de datos asi como su precio por litro, la cantidad de litros y si asi lo deciden un descuento

El tick se abre en otra ventana y también las ventas por día. También podemos consultar los registros por fecha o borrar un registro especifico

Podemos visualizar tambien las ventas totales, ver la cantidad de ganancias y borrar un registro desde este apartado
