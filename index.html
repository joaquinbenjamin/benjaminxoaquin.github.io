Portafolio java

En el lenguaje java los programas se ejecutan sobre la máquina virtual de java, es decir son interpretados, pero esto agrega una capa más de compatibilidad, haciendo posible que los programas se puedan ejecutar en cualquier ordenador independientemente de su arquitectura.
